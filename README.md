# Transaction-Data-Analysis

This project analyzes a retail transaction dataset to extract business insights related to sales, customers, products, services, and geography.

### Objective
The goal of this analysis is to understand:
- Sales trends over time
- Customer purchasing behavior
- Product and service performance
- Geographic sales distribution
- Payment method patterns
- High-value customer and market segments

### Approach
The analysis was performed using **Python (Pandas)** in a Jupyter Notebook.

Key steps:
1. Loaded and explored the transaction dataset
2. Cleaned and converted date fields for time analysis
3. Aggregated sales by month, quarter, state, city, product, and service
4. Computed KPIs such as:
   - Total sales
   - Average transaction value
   - Repeat customer rate
   - Category revenue contribution
5. Identified high-performing and underperforming segments
6. Derived business recommendations for inventory and marketing focus

### Insights Generated
The notebook answers 35 business questions, including:
- Highest sales month and quarter
- Top-revenue products and services
- Customer spending patterns
- Geographic sales leaders
- Payment behavior differences
- Seasonal demand trends

### Tools Used
- Python
- Pandas
- Jupyter Notebook
